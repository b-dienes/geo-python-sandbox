# Geo Python Sandbox

## **Background & Motivation**

This repository serves as a workspace for ongoing engagement with geospatial Python tools, demonstrating workflows, techniques, and scripts across common GIS Python libraries.

## **Technology Stack**

A collection of geospatial Python experiments and utilities using:

- GeoPandas
- Rasterio
- PyProj


---

## **Architecture / Project Structure**

```text
geo-python-sandbox/
├── data/
│   ├── inputs/       # Place input datasets here (e.g., GeoPackage files)
│   └── outputs/      # Place outputs from scripts here (e.g., cleaned GeoDataFrames, exports)
├── src/
│   ├── geopandas_demo.py       # main GeoPandas practice script
│   └── utils/
│       └── paths.py            # helper functions for file paths
├── README.md
└── .gitignore
```
